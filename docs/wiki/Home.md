# AI News Monitor Wiki

AI News Monitor is a local-first desktop app for monitoring public news sources and sending source-grounded alerts through user-configured channels.

## Start Here

- [Getting Started](Getting-Started.md)
- [Configuration](Configuration.md)
- [Troubleshooting](Troubleshooting.md)

## What It Does

- Monitors public RSS/Atom feeds and supported public source candidates.
- Ranks articles against user-defined topics.
- Uses the user's OpenAI-compatible LLM for relevance checks, summaries, translation, and optional analysis.
- Groups related articles into event-level alerts.
- Sends alerts through email, Telegram, webhook, or supported relay channels.

## What It Does Not Do

- It does not trade, place orders, connect to broker APIs, or provide investment advice.
- It does not store API keys in the repository.
- It does not require private or paywalled sources.
