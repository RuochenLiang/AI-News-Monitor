Use this page as the canonical source for the repository sidebar, release notes, and project description.

## Short Description

Local-first desktop AI news monitor with source-grounded relevance scoring, OpenAI-compatible LLM summaries, and phone-friendly alerts.

## Longer Description

AI News Monitor is a Python desktop app for monitoring public Chinese and English news sources. Users configure their own OpenAI-compatible LLM, source packages, topics, and notification channels. The app ranks source-grounded articles, groups related articles into events, verifies relevance, and sends concise alerts through email, Telegram, webhook, or relay channels.

It is not a trading bot, broker integration, stock recommender, or investment adviser.
