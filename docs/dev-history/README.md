# Development History

This directory contains historical Codex prompts and planning notes used while preparing AI News Monitor for release.

These files are not required for normal installation, use, or GitHub Actions. They are kept here only as project history so the repository root stays focused on source code, documentation, configuration examples, and release metadata.

The historical prompts have been consolidated into [prompt.md](prompt.md). That file keeps the original sequence as numbered sections from `01` through `11`.
